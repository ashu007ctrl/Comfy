                    AI STRESS DETECTION APPLICATION - "COMFY"
================================================================================

PROJECT OVERVIEW
----------------
Comfy is a full-stack AI-powered stress detection web application that analyzes 
user responses through an interactive questionnaire and provides personalized 
stress management recommendations. The app uses Google's Gemini AI to generate 
accurate stress assessments and actionable tips.


TECH STACK
----------

Frontend:
  - React.js (v18+) - Component-based UI library
  - Vite - Fast build tool and development server
  - Framer Motion - Advanced animations and transitions
  - Axios - HTTP client for API requests
  - Lucide React - Modern icon library
  - React Helmet Async - Dynamic meta tag management for SEO
  - CSS3 - Custom styling with gradients, glassmorphism, animations

Backend:
  - Node.js - JavaScript runtime
  - Express.js - Web application framework
  - Google Generative AI (Gemini API) - AI-powered stress analysis
  - CORS - Cross-origin resource sharing
  - dotenv - Environment variable management

APIs & Services:
  - Google Gemini API - AI text generation for stress analysis
  - Web3Forms API - Contact form handling

ARCHITECTURE
------------

Client-Server Architecture:
  - Frontend (Port 5173): React SPA served by Vite
  - Backend (Port 3000): Express.js REST API
  - Communication: JSON over HTTP (Axios)

Project Structure:
  /client
    /src
      /assets - Images and static files
      /components - Reusable UI components
      /data - Question data
      /pages - Page-level components
      App.jsx - Main application logic
      index.css - Global styles
  
  /server
    server.js - Express server + Gemini AI integration
    .env - Environment variables (API keys)


API ENDPOINTS
-------------

POST /api/analyze-stress
  Request: { answers: { questionId: value, ... } }
  Response: { 
    score: number, 
    level: string, 
    tips: string[],
    analysis: string 
  }


WORKFLOW
--------

1. User lands on Home page
2. Clicks "Start Free Test" → Questionnaire loads
3. Answers 10 questions with Back/Next navigation
4. Submits answers → Loading state
5. Backend processes via Gemini AI
6. Results page displays with score, level, tips
7. User can retake or navigate back to home
